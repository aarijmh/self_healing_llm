## ⚡ Quick Start (3 Steps)

### Step 1: Activate Environment
```bash
# Windows
..\.venv\Scripts\activate

# Linux/Mac
source ../.venv/bin/activate
```

### Step 2: Run Demo
```bash
# Windows
.\start_demo.bat

# Linux/Mac
chmod +x start_demo.sh
./start_demo.sh
```

### Step 3: Open Browser
Browser will open automatically to: **http://localhost:8000**
